텍스트 프롬프트 구조

  1. 역할: AI가 어떤 전문가처럼 답할지 지정
  2. 작업: 분석, 생성, 검토, 수정처럼 동사를 명확히 지정
  3. 맥락: 프로젝트 상황, 오류 메시지, 사용 언어 제공
  4. 형식: 표, 체크리스트, JSON, 단계별 설명 등 지정
  5. 제약: 분량, 수준, 금지사항, 대상 독자 지정

컨텍스트 엔지니어링

  • 필요한 파일, 오류 메시지, 실행 환경만 선별해서 제공
  • 관련 없는 정보와 오래된 조건은 제거해 신호대잡음비를 높이기
  • 작업 종류에 따라 배경 자료, 예시, 출력 형식을 동적으로 바꾸기
  • 실행 → 검증 → 수정 과정을 반복해 맥락을 개선하기
